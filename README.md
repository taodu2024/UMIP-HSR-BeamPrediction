# UMIP-HSR-BeamPrediction
This repository provides the simulation framework and model implementation
for the paper:
**"UMIP-Enabled Time-Aware Multimodal LLM Framework for Predictive Beamforming in High-Speed Railway Networks"**

## Status

The repository is currently under preparation.
Full implementation and trained models will be released upon paper acceptance.

## Simulation Platform

Multimodal data generation is conducted based on the AirSim platform:

https://github.com/microsoft/AirSim

## Environment

- Python 3.10
- PyTorch 2.1+
- CUDA 12.8

More details will be provided in future updates.
